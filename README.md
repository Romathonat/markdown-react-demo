# markdown-react-demo

This demo shows you a way of using markdown files with ReactJS. 
## How to use ?
All you need is npm.

Then :
``` bash
  git clone https://github.com/Romathonat/markdown-react-demo.git
  cd markdown-react-demo
  npm install
  npm start
```

Then go to [http://localhost:3000/](http://localhost:3000/)
